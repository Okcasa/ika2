# **App Name**: LeadSorter Pro

## Core Features:

- CSV Upload: Allow users to upload a CSV file containing lead data.
- Data Parsing: Parse the CSV file and extract relevant data such as business name, phone number, and website.
- Data Validation: Validate the extracted data, checking for missing or invalid information. It uses tool calls for address or phone number verifications
- Business Type Classification: Use AI to classify the type of business based on available information, user confirmation, or user specification if needed.
- Data Correction and Deduplication: Correct any errors or inconsistencies in the data, and remove duplicate entries. This will happen automatically. It uses tool calls for finding updated URLs if available.
- Data Display and Editing: Display the sorted and corrected lead data in a table format, allowing users to manually edit and confirm the information.
- Download Corrected List: Allow users to download the sorted and corrected lead data in CSV format.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) for a professional and trustworthy feel.
- Background color: Light gray (#F5F5F5) to provide a clean and modern backdrop.
- Accent color: Green (#2ECC71) for positive actions and confirmations.
- Body and headline font: 'Inter' sans-serif for a modern and readable style.
- Code font: 'Source Code Pro' for displaying data or information which looks extracted
- Use clear and concise icons for common actions like upload, download, edit, and delete.
- Employ a clean and structured layout with clear sections for data upload, display, and download.
- Subtle animations to indicate data loading and processing states.
